## Project Development Description  

The main purpose of this Task Management System (TMS) is to help users manage their daily tasks in a simple and organized way. While building this project, I focused on creating a basic but functional application where users can add tasks, assign priority levels such as High, Medium, and Low, and track their progress by moving them from the “To Do” section to the “Completed” section.  

To build this project, I first created the structure using HTML. I designed the layout with an input field for adding tasks, a dropdown menu for selecting priority, and two main sections to display tasks: one for pending tasks and another for completed tasks. This helped me organize the content clearly and make the application easy to use.  

After setting up the structure, I used CSS to style the application. I focused on keeping the design simple and clean by adding spacing, background colors, and basic styling. I also implemented color coding for task priorities, such as red for high priority, orange for medium, and green for low, which makes it easier to identify task importance visually.  

Next, I added functionality using JavaScript. I created functions to handle user interactions such as adding a new task, marking a task as completed, and deleting a task. When a user adds a task, it is displayed in the “To Do” section with the selected priority. When the “Done” button is clicked, the task moves to the completed section, and when the “Delete” button is clicked, the task is removed from the list. This allowed me to understand how to dynamically update the content of a webpage using JavaScript.  

While developing this project, I focused on keeping the code simple, organized, and easy to understand. I made sure that HTML, CSS, and JavaScript worked together properly to create a smooth and functional user experience.  

Overall, this project helped me gain practical experience in front-end development and improved my understanding of how structure, design, and functionality are combined to build a working web application.
